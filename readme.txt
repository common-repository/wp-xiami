== Description ==

WP-Xiami 虾米音乐同步：支持Html5+Flash的虾米音乐播放器，支持同步个人虾米专辑和精选集。


== Installation ==

1. 上传 `wp-xiami`目录 到 `/wp-content/plugins/` 目录
2. 在后台插件菜单激活该插件
